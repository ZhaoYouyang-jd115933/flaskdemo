About This Flask + Wikipedia App

Welcome to the Wikipedia Searcher! This simple web application allows you to search for topics on Wikipedia and view a clean summary of results. It's designed to be beginner-friendly and demonstrate how Flask, templates, and APIs work together.

Features

  • Search Wikipedia and display summary results
  • Clean, responsive interface styled with CSS
  • Uses Flask and Jinja2 templating
  • Handles search errors and invalid queries

Technologies Used

  • Python 3
  • Flask Web Framework
  • Wikipedia API
  • HTML / CSS + Font Awesome
